# call-center-app

### Purpose:
	Dashboard which shows Data Reporting/Analytics charts and Basic Data CRUD Forms

### List of Features:
* Cards
* Area Graph and Bar Graph
* Word cloud
* Timeline
* Data Tables with search, sort and pagination
* Form Wizard
* Date Picker
* Form Validations
* Internationalization (Manual and Google)
